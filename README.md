# Sports Tournament Platform

Single-page marketing site and tournaments directory for Indian sports organizers. Built with Vite, React (TypeScript), Tailwind CSS, shadcn/ui, TanStack Query, and Framer Motion.

## Local Development

```bash
git clone <repo-url>
cd <project-directory>
npm install
npm run dev
```

## Available Scripts

- `npm run dev` – start the dev server with hot reload
- `npm run build` – build the production bundle
- `npm run preview` – preview the production build
- `npm run lint` – run ESLint across the project

## Deployment

Run `npm run build` and deploy the `dist/` output to any static host (Netlify, Vercel, Cloudflare Pages, etc.). Set the site to serve `index.html` for SPA routing.
